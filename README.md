# book-spider
基于cheerio+superagent+async+mysql的小说爬虫。
------------------------------------------------------------------
### 下载
```
git clone https://github.com/biyunbo/book.git
cd book
npm install
``` 

### 启动
```
(要安装有mysql)
node book.js
node list.js
```
