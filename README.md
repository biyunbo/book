# react-cnode
基于cheerio+superagent+async+mysql
------------------------------------------------------------------
### 下载
```
git clone https://github.com/biyunbo/react-cnode.git
cd book
npm install
yarn(安装依赖)
``` 

### 启动
```
node book.js
node list.js
```
